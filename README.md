🏥 Healthcare Analytics & Patient Insights Dashboard
Overview
This project showcases an interactive Power BI dashboard built to analyze a healthcare dataset containing over **55,000 patient records**. The dashboard transforms raw healthcare data into meaningful business insights using data cleaning, DAX calculations, KPIs, and interactive visualizations.

🎯 Objectives
- Analyze patient demographics
- Monitor hospital revenue
- Compare billing across admission types
- Identify common medical conditions
- Understand patient distribution across age groups
- Build an interactive dashboard for business decision-making

📊 Dataset
- Records: 55,000+
- Domain: Healthcare
- Format: Excel (.xlsx)

Columns Used
- Patient Name
- Age
- Gender
- Blood Type
- Medical Condition
- Admission Type
- Hospital
- Insurance Provider
- Billing Amount
- Date of Admission
- Discharge Date
- Medication
- Test Results

🛠 Tools & Technologies
- Power BI Desktop
- Power Query
- DAX
- Microsoft Excel

📈 Dashboard Features
 KPI Cards
- Average Billing
- Average Stay (Days)
- Total Patients
- Total Revenue

Visualizations
- Average Stay by Age Group & Admission Type
- Top 10 Hospitals by Revenue
- Average Billing by Admission Type
- Top Medical Conditions by Patient Count
- Patient Distribution by Age Group & Gender

Interactive Filters
- Gender
- Blood Type
- Insurance Provider

💡 Key Insights
- Average patient billing is around ₹25K.
- Total revenue exceeds ₹1.4 Billion.
- Average hospital stay is approximately 16 days.
- Arthritis, Diabetes, and Hypertension are among the most common medical conditions.
- Revenue and billing vary across hospitals and admission types.
